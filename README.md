# 30 Days Data Science Challenge

Completing 30 data science projects in 30 days using
SQL, Excel, Power BI, Python, and Machine Learning.

Every project answers a real business question with a
final insight — not just practice, but actual findings.

---

## Tools Used
- SQL (SQLiteOnline / MySQL Workbench)
- Microsoft Excel
- Power BI
- Python (Pandas, Matplotlib, Scikit-learn)
- Machine Learning (Scikit-learn, XGBoost)
