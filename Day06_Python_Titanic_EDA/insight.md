## Day 6 — Python Titanic EDA

**Dataset:** Titanic (891 rows, 11 columns after cleaning)
**Question:** Did gender or class determine survival?

**Answer:**
- Female survival: 74.2% vs Male: 18.9%
- Women were 4x more likely to survive!
- Class 1 survival: 62.9% vs Class 3: 24.2%
- Class 1 passengers were 2.6x more likely to survive!

**Both gender AND class determined survival!**

**Tool:** Python (Pandas, Matplotlib)
**Key functions:** groupby(), mean(), plot()
